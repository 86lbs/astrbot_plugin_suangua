## [v4.5.0] - 2026-03-02

### 新增配置项

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 显示爻辞 | ✅ 开启 | 是否在卦象结果中显示爻辞 |
| 显示运势指引 | ✅ 开启 | 是否在卦象结果中显示运势指引 |
| AI解卦默认提示词 | (见配置) | 当会话未设置人格时使用的系统提示词 |
| AI解卦等待提示 | (见配置) | AI解卦时显示的等待提示语，支持 `{卦名}` 占位符 |
| 显示AI解卦提示 | ✅ 开启 | 是否在算卦结果末尾显示AI解卦提示 |

### 配置项总览

| 配置项 | 类型 | 说明 |
|--------|------|------|
| 启用变卦 | bool | 是否启用变卦功能 |
| 显示起卦过程 | bool | 是否显示每次抛掷铜钱的结果 |
| 显示爻辞 | bool | 是否显示爻辞 |
| 显示运势指引 | bool | 是否显示运势指引 |
| AI解卦使用T2I | bool | AI解卦结果是否转图片 |
| AI解卦默认提示词 | string | 默认系统提示词 |
| AI解卦等待提示 | string | 等待提示语，支持 `{卦名}` |
| 显示AI解卦提示 | bool | 是否显示AI解卦提示 |
