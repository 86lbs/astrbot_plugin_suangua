# 更新日志

## [1.2.0] - 2025-01-XX

### 新增

- **指定 LLM 提供商**：新增 `ai_divine_provider_id` 配置项，支持指定 AI 解卦使用的 LLM 提供商 ID
- **指定人格**：新增 `ai_divine_persona_name` 配置项，支持指定 AI 解卦使用的人格名称
- **自定义默认提示词**：新增 `ai_divine_default_prompt` 配置项，支持自定义默认系统提示词
- **配置日志**：AI 解卦时会记录使用的 Provider 和人格来源，便于调试

### 改进

- 优化 `_get_ai_interpretation` 方法的 Provider 和人格获取逻辑
- 完善错误处理，当指定的 Provider 或人格不存在时自动回退到默认配置
- 更新帮助信息，添加新增配置项说明

### 技术细节

- Provider 获取优先级：插件配置 → 当前会话配置
- 人格获取优先级：插件配置 → 当前会话配置 → 默认提示词
- 使用 `provider_manager.get_provider_by_id()` 获取指定 Provider
- 从 `persona_manager.personas_v3` 列表中查找指定人格

## [1.1.0] - 2024-XX-XX

### 新增

- 支持引用消息进行 AI 解卦
- 添加算卦结果缓存机制（10分钟有效期）
- 支持从引用内容解析卦象信息

### 改进

- 优化引用消息检测逻辑
- 添加用户级缓存隔离（同一群聊不同用户独立缓存）

## [1.0.0] - 2024-XX-XX

### 新增

- 传统金钱卦起卦法
- 六十四卦完整数据
- 变卦支持
- LLM 工具调用
- 指令调用
- AI 解卦功能
- 可配置选项
