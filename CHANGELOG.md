## [v4.1.4] - 2026-03-01

### 新增
- 🎭 AI解卦使用当前会话的人格（Persona）系统提示词
- ⚙️ 添加「AI解卦使用T2I」配置项，默认开启

### 修复
- 🖼️ AI解卦结果可根据配置选择是否使用T2I

### 配置项
| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 启用变卦 | ✅ 开启 | 关闭后不产生变爻，只显示本卦 |
| 显示起卦过程 | ❌ 关闭 | 开启后显示每次抛掷铜钱的结果 |
| AI解卦使用T2I | ✅ 开启 | AI解卦结果是否使用文本转图片功能 |

## [v4.1.0] - 2026-03-01

### 新增
- 🪙 采用传统金钱卦起卦法，模拟抛掷三枚铜钱六次确定卦象
- 根据老阳(三正)/老阴(三反)自然产生变爻
- 真实还原传统起卦方式

### 修复
- 变卦逻辑修正：变爻不再随机选择，而是根据起卦结果自然产生
- 老阳变阴，老阴变阳，符合易经原理

### 变更
- 移除独立的变卦指令
- 算卦默认带变卦（可通过配置关闭）

---

## [v4.0.0] - 2026-03-01

### 新增
- 重构插件，修复指令调用问题
- 使用 `event.set_result()` 替代 `yield` 方式返回结果
- 添加英文指令名，中文作为别名
- 同时支持指令调用和 LLM 工具调用

### 指令列表
| 指令 | 别名 | 说明 |
|------|------|------|
| `/suangua` | `/算卦` `/算一卦` `/占卜` | 生成卦象 |
| `/aijiiegua` | `/ai解卦` `/AI解卦` | AI详细解读 |
| `/guaxiang 卦名` | `/卦象` `/查卦` | 查询特定卦象 |
| `/liushisigua` | `/六十四卦` `/卦列表` | 列出所有卦象 |

## 版本命名规则

- **主版本号 (Major)**: 不兼容的 API 更改
- **次版本号 (Minor)**: 向后兼容的功能新增
- **修订号 (Patch)**: 向后兼容的问题修复
