# 易经算卦插件 🧙‍♂️

AstrBot 易经六十四卦算卦插件，采用传统金钱卦起卦法，支持指令调用和 LLM 函数工具调用。

## 功能特点

- 🪙 **传统金钱卦起卦法** - 模拟抛掷三枚铜钱六次确定卦象
- ☯️ **变卦支持** - 根据老阳老阴自然产生变爻
- 🤖 **LLM 工具调用** - 支持 AI 自动调用算卦
- 🎭 **人格支持** - AI 解卦使用当前会话的人格
- 🔧 **指定 LLM 提供商** - 可配置 AI 解卦使用的特定 LLM 提供商
- 🎯 **指定人格** - 可配置 AI 解卦使用的特定人格
- ⚙️ **动态配置** - 支持通过指令动态切换 LLM 和人格
- ⚡ **可配置** - 支持多种配置选项

## 安装

### 方法一：插件市场安装

1. 打开 AstrBot 管理面板
2. 进入「插件」→「插件市场」
3. 搜索「易经算卦」
4. 点击安装

### 方法二：手动安装

1. 下载插件：[Releases](https://github.com/86lbs/astrbot_plugin_suangua/releases)
2. 解压到 `data/plugins/astrbot_plugin_suangua/` 目录
3. 重启 AstrBot

## 使用方法

### 指令调用

| 指令 | 别名 | 说明 |
|------|------|------|
| `/算一卦 [问题]` | - | 生成卦象 |
| `/AI解卦` | `/ai解卦` | AI 详细解读（需引用算卦结果） |
| `/算卦设置` | `/卦设置` | 查看和设置 LLM/人格 |
| `/卦象 <卦名>` | `/查卦` | 查询特定卦象信息 |
| `/六十四卦` | `/卦列表` | 列出所有卦象 |
| `/算卦帮助` | `/卦帮助` | 显示帮助信息 |

> 💡 `/` 为默认唤醒词，可在 AstrBot 管理面板中修改

### 无唤醒词触发

直接发送以下消息也可触发算卦：
- `算卦`
- `算卦 今天运势`
- `算一卦`
- `算一卦 事业`

### LLM 工具调用

当用户说「帮我算一卦」、「算一下」等，AI 会自动调用算卦工具。

### 使用示例

#### 基础算卦

```
用户：/算一卦
机器人：
【乾卦】
━━━
━━━
━━━
━━━
━━━
━━━
卦性：纯阳
含义：天行健，君子以自强不息
...
```

#### AI 解卦

1. 先发送 `/算一卦` 获取卦象
2. 长按卦象消息，选择「引用」
3. 发送 `/AI解卦`
4. AI 会根据配置的人格进行详细解读

## 配置项

在 AstrBot 管理面板 → 插件 → 易经算卦 → 设置 中配置：

### 基础配置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| 启用变卦 | bool | ✅ 开启 | 是否启用变卦功能 |
| 显示起卦过程 | bool | ❌ 关闭 | 是否显示每次抛掷铜钱的结果 |
| 显示爻辞 | bool | ✅ 开启 | 是否显示爻辞 |
| 显示运势指引 | bool | ✅ 开启 | 是否显示运势指引 |
| AI解卦使用T2I | bool | ✅ 开启 | AI解卦结果是否转图片 |
| AI解卦等待提示 | string | (预设) | 等待提示语，支持 `{卦名}` 占位符 |
| 显示AI解卦提示 | bool | ✅ 开启 | 是否在算卦结果末尾显示提示 |

### 高级配置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| AI解卦LLM提供商ID | string | 空 | 指定AI解卦使用的LLM提供商ID，留空则使用当前会话配置 |
| AI解卦人格名称 | string | 空 | 指定AI解卦使用的人格名称，留空则使用当前会话人格 |
| AI解卦默认提示词 | string | (预设) | 当未指定人格或人格不存在时使用的默认系统提示词 |

> 💡 **提示**：LLM提供商ID可在 AstrBot 管理面板的「配置」→「服务提供商配置」中查看

## 动态配置（/算卦设置）

使用 `/算卦设置` 指令可以动态查看和切换配置：

| 指令 | 说明 |
|------|------|
| `/算卦设置` | 查看当前配置和可用选项 |
| `/算卦设置 provider <ID>` | 设置 LLM 提供商 |
| `/算卦设置 persona <名称>` | 设置人格 |
| `/算卦设置 重置` | 恢复默认配置 |

**示例**：
```
/算卦设置
/算卦设置 provider zhipu/glm-4.5-air
/算卦设置 persona 算命大师
/算卦设置 重置
```

> ⚠️ **注意**：通过指令设置的配置为临时生效，重启 AstrBot 后会恢复为配置文件中的值。

## 指定 LLM 提供商和人格

### 使用场景

- 希望 AI 解卦使用特定的模型（如 GPT-4、Claude 等）
- 希望 AI 解卦使用特定的人格风格（如「算命大师」、「易经专家」等）
- 不同会话使用统一的解卦风格

### 配置方法

**方法一：管理面板配置（永久生效）**

1. 打开 AstrBot 管理面板
2. 进入「插件」→「易经算卦」→「设置」
3. 在「AI解卦LLM提供商ID」中填写要使用的提供商ID
4. 在「AI解卦人格名称」中填写要使用的人格名称
5. 保存配置

**方法二：指令配置（临时生效）**

```
/算卦设置 provider zhipu/glm-4.5-air
/算卦设置 persona 算命大师
```

### 优先级说明

AI解卦获取 Provider 和人格的优先级：

1. **Provider 优先级**：
   - 插件配置的「AI解卦LLM提供商ID」→ 当前会话配置的 Provider

2. **人格优先级**：
   - 插件配置的「AI解卦人格名称」→ 当前会话配置的人格 → 默认提示词

## 金钱卦起卦法

本插件采用传统金钱卦起卦法：

1. **抛掷铜钱** - 每次抛掷三枚铜钱
2. **确定爻象** - 正面为阳(3)，反面为阴(2)
3. **计算总和** - 6=老阴(变)，7=少阳，8=少阴，9=老阳(变)
4. **六爻成卦** - 从初爻到上爻，共抛掷六次
5. **变爻转化** - 老阳变阴，老阴变阳

## 六十四卦

插件包含完整的六十四卦数据，每个卦象包含：

- 卦象符号（如：☰☷）
- 卦性（如：纯阳、纯阴）
- 基本含义
- 六爻爻辞

## 常见问题

### Q: 指令不触发？

A: 确保使用唤醒前缀（如 `/`）或 @机器人 后发送指令。也可以直接发送「算卦」或「算一卦」（无唤醒词）。

### Q: AI 解卦不使用人格？

A: 检查插件配置中的「AI解卦人格名称」是否正确，或确保当前会话已设置人格。

### Q: 指定的 LLM 提供商不生效？

A: 确保填写的提供商ID正确，可在 AstrBot 管理面板的「服务提供商配置」中查看已配置的提供商ID。也可以使用 `/算卦设置` 查看可用的提供商列表。

### Q: 引用消息无法识别？

A: 请使用消息平台的「引用」功能，而不是复制粘贴文本。

### Q: 如何查看可用的 LLM 提供商？

A: 发送 `/算卦设置` 或 `/算卦帮助`，会显示当前可用的 LLM 提供商列表及其 ID。
