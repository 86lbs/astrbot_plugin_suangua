# AstrBot 算卦插件 🔮

一个基于易经六十四卦的 [AstrBot](https://astrbot.app) 聊天机器人插件，支持指令调用和 AI 函数工具调用两种触发方式。

## 免责声明

插件开发全程均由 AI 完成，仅供娱乐参考，请勿迷信。

## 功能特点

- 🤖 **AI 函数工具调用**：作为 LLM 工具供 AI 调用，实现智能算卦
- 💬 **指令调用**：支持多种指令触发方式
- 🎲 **随机起卦**：随机生成六十四卦中的某一卦
- 🔄 **变卦功能**：支持变卦，展示事物发展变化
- 📖 **完整卦象信息**：包含卦名、卦象、性质、含义、爻辞、运势指引
- 🧠 **AI 解卦**：引用卦象结果进行 AI 详细解读

## 安装方式

### 方法一：通过 AstrBot 插件市场安装

1. 打开 AstrBot 管理面板
2. 进入插件市场
3. 搜索关键词 "算卦" 或 "suangua"
4. 安装

### 方法二：手动安装

```bash
# 在 AstrBot 的 data/plugins 目录下
git clone https://github.com/86lbs/astrbot_plugin_suangua.git
```

然后重启 AstrBot。

## 使用方法

### 一、指令调用方式

用户需要使用唤醒前缀（如 `/`）或 At 机器人后发送指令：

| 指令 | 别名 | 说明 |
|------|------|------|
| `/suangua` | `/算卦` `/算一卦` `/占卜` | 生成卦象 |
| `/biangua` | `/变卦` `/变一卦` | 生成带变卦的结果 |
| `/aijiiegua` | `/ai解卦` `/AI解卦` | AI详细解读（需引用卦象结果） |
| `/guaxiang 卦名` | `/卦象` `/查卦` | 查询特定卦象信息 |
| `/liushisigua` | `/六十四卦` `/卦列表` | 列出所有卦象 |

### 二、AI 自然对话触发

用户无需记忆任何指令，直接与 AI 对话即可：

```
用户: 帮我算一卦
AI: [自动调用 divine_hexagram 工具]
    【乾卦】
    ━━━
    ━━━
    ━━━
    ...
```

### 支持的对话方式

- "帮我算一卦"
- "我想知道我的事业运势"
- "帮我占卜一下感情"
- "今天运势怎么样"

AI 会自动识别用户意图并调用算卦工具。

## 输出内容

每次算卦会输出以下信息：

| 内容 | 说明 |
|------|------|
| 卦名 | 六十四卦之一 |
| 卦象 | 六爻图形显示 |
| 卦性 | 卦的基本性质 |
| 含义 | 卦的基本含义 |
| 爻辞 | 随机选取的一爻爻辞 |
| 变卦 | （可选）变卦信息 |
| 运势指引 | 事业、感情、财运、健康等方面的建议 |

## 技术实现

- **LLM 工具**：使用 `@llm_tool` 装饰器注册为 AI 可调用的函数工具
- **指令调用**：使用 `@filter.command` 装饰器注册指令，支持中文别名
- **随机起卦**：从六十四卦中随机选择一卦
- **变卦算法**：随机生成变爻位置，计算变卦结果
- **本地数据**：包含完整的六十四卦 JSON 数据
