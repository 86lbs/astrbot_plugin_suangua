# 易经算卦插件 🧙‍♂️

AstrBot 易经六十四卦算卦插件，采用传统金钱卦起卦法，支持指令调用和 LLM 函数工具调用。

## 功能特点

- 🪙 **传统金钱卦起卦法** - 模拟抛掷三枚铜钱六次确定卦象
- ☯️ **变卦支持** - 根据老阳老阴自然产生变爻
- 🤖 **LLM 工具调用** - 支持 AI 自动调用算卦
- 🎭 **人格支持** - AI 解卦使用当前会话的人格
- ⚙️ **可配置** - 支持多种配置选项

## 安装

### 方法一：插件市场安装

1. 打开 AstrBot 管理面板
2. 进入「插件」→「插件市场」
3. 搜索「易经算卦」
4. 点击安装

### 方法二：手动安装

1. 下载插件：[Releases](https://github.com/86lbs/astrbot_plugin_suangua/releases)
2. 解压到 `data/plugins/astrbot_plugin_suangua/` 目录
3. 重启 AstrBot

## 使用方法

### 指令调用

| 指令 | 别名 | 说明 |
|------|------|------|
| `[唤醒词]算卦` | `[唤醒词]算一卦` | 生成卦象 |
| `[唤醒词]AI解卦` | `[唤醒词]ai解卦` | AI 详细解读（需引用算卦结果） |
| `[唤醒词]卦象 卦名` | `[唤醒词]查卦` | 查询特定卦象信息 |
| `[唤醒词]六十四卦` | `[唤醒词]卦列表` | 列出所有卦象 |
| `[唤醒词]算卦帮助` | `[唤醒词]卦帮助` `[唤醒词]帮助算卦` | 显示帮助信息 |

> 💡 `[唤醒词]` 默认为 `/`，可在 AstrBot 管理面板中修改

### LLM 工具调用

当用户说「帮我算一卦」、「算一下」等，AI 会自动调用算卦工具。

### 使用示例

#### 基础算卦

```
用户：/算卦
机器人：
【乾卦】
━━━
━━━
━━━
━━━
━━━
━━━
卦性：纯阳
含义：天行健，君子以自强不息
...
```

#### AI 解卦

1. 先发送 `/算卦` 获取卦象
2. 长按卦象消息，选择「引用」
3. 发送 `/AI解卦`
4. AI 会根据当前人格进行详细解读

## 配置项

在 AstrBot 管理面板 → 插件 → 易经算卦 → 设置 中配置：

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| 启用变卦 | bool | ✅ 开启 | 是否启用变卦功能 |
| 显示起卦过程 | bool | ❌ 关闭 | 是否显示每次抛掷铜钱的结果 |
| 显示爻辞 | bool | ✅ 开启 | 是否显示爻辞 |
| 显示运势指引 | bool | ✅ 开启 | 是否显示运势指引 |
| AI解卦使用T2I | bool | ✅ 开启 | AI解卦结果是否转图片 |
| AI解卦等待提示 | string | (预设) | 等待提示语，支持 `{卦名}` 占位符 |
| 显示AI解卦提示 | bool | ✅ 开启 | 是否在算卦结果末尾显示提示 |

> 💡 AI解卦会自动使用当前会话的人格，无需额外配置

## 金钱卦起卦法

本插件采用传统金钱卦起卦法：

1. **抛掷铜钱** - 每次抛掷三枚铜钱
2. **确定爻象** - 正面为阳(3)，反面为阴(2)
3. **计算总和** - 6=老阴(变)，7=少阳，8=少阴，9=老阳(变)
4. **六爻成卦** - 从初爻到上爻，共抛掷六次
5. **变爻转化** - 老阳变阴，老阴变阳

## 六十四卦

插件包含完整的六十四卦数据，每个卦象包含：

- 卦象符号（如：☰☷）
- 卦性（如：纯阳、纯阴）
- 基本含义
- 六爻爻辞

## 常见问题

### Q: 指令不触发？

A: 确保使用唤醒前缀（如 `/`）或 @机器人 后发送指令。

### Q: AI 解卦不使用人格？

A: 确保当前会话已设置人格，AI 解卦会使用会话的人格系统提示词。

### Q: 引用消息无法识别？

A: 请使用消息平台的「引用」功能，而不是复制粘贴文本。
