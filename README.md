# AstrBot 算卦插件 🔮

一个基于易经六十四卦的 [AstrBot](https://astrbot.app) 聊天机器人插件，采用**传统金钱卦起卦法**，支持指令调用和 AI 函数工具调用两种触发方式。

## 免责声明

插件开发全程均由 AI 完成，仅供娱乐参考，请勿迷信。

## 功能特点

- 🪙 **传统金钱卦起卦法**：模拟抛掷三枚铜钱六次，真实还原传统起卦方式
- 🔄 **变卦功能**：根据老阳/老阴自然产生变爻，展示事物发展变化
- 🤖 **AI 函数工具调用**：作为 LLM 工具供 AI 调用，实现智能算卦
- 💬 **指令调用**：支持多种指令触发方式
- ⚙️ **可配置**：支持在管理面板配置变卦开关等选项

## 安装方式

### 方法一：通过 AstrBot 插件市场安装

1. 打开 AstrBot 管理面板
2. 进入插件市场
3. 搜索关键词 "算卦" 或 "suangua"
4. 安装

### 方法二：手动安装

```bash
# 在 AstrBot 的 data/plugins 目录下
git clone https://github.com/86lbs/astrbot_plugin_suangua.git
```

然后重启 AstrBot。

## 传统金钱卦起卦法

### 起卦原理

本插件采用传统**金钱卦**起卦法，模拟抛掷三枚铜钱六次来确定卦象：

| 铜钱组合 | 总和 | 名称 | 爻象 | 是否变爻 |
|---------|------|------|------|----------|
| 三反(2+2+2) | 6 | 老阴 | ━ ━ | ✅ 变为阳 |
| 二反一正 | 7 | 少阳 | ━━━ | ❌ 不变 |
| 二正一反 | 8 | 少阴 | ━ ━ | ❌ 不变 |
| 三正(3+3+3) | 9 | 老阳 | ━━━ | ✅ 变为阴 |

### 变卦规则

- **老阳（三正）**：阳爻变阴爻，标记为 ○
- **老阴（三反）**：阴爻变阳爻，标记为 ×
- **少阳、少阴**：不变

## 插件配置

在 AstrBot 管理面板中可以配置以下选项：

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 启用变卦 | 开启 | 是否启用变卦功能，关闭后不产生变爻 |
| 显示起卦过程 | 关闭 | 是否显示每次抛掷铜钱的结果 |
